Mahara, Application Mobile de Développement de Compétences


1. Introduction
c'est une application mobile développée avec React Native visant à accompagner les utilisateurs dans le développement de différentes compétences à travers des plans personnalisés, un suivi de progression et un système de motivation basé sur la gamification.


2. Objectifs du Projet
Objectif principal : Concevoir et développer une application mobile multiplateforme capable de générer des plans hebdomadaires personnalisés et d’adapter leur difficulté en fonction des performances de l’utilisateur.


Objectifs spécifiques :
- Implémenter un système modulaire multi-compétences
- Concevoir un générateur automatique de plans
- Mettre en place un système de suivi de progression
- Intégrer un système de motivation (streak, points XP)
- Concevoir une interface utilisateur moderne et intuitive


3. Compétences Supportées
1. Musique : gammes, rythme, morceaux, improvisation
2. Photographie : composition, lumière, portrait, retouche
3. Fitness : renforcement musculaire, cardio, mobilité, récupération
4. Langues : vocabulaire, compréhension orale, expression écrite, révision
Le système sera conçu de manière modulaire afin de permettre l’ajout futur de nouvelles compétences.


4. Fonctionnalités Principales
- Création et gestion du profil utilisateur
- Sélection de la compétence et du niveau
- Génération automatique d’un plan hebdomadaire
- Suivi des tâches complétées
- Système de streak (jours consécutifs)
- Attribution de points d’expérience (XP)
- Adaptation automatique de la difficulté selon les performances
- Tableau de bord statistique


5. Architecture Technique
Framework : React Native
Langage : JavaScript / TypeScript
Architecture : séparation en couches (UI – Logique métier – Services – Données)
Cette architecture garantira la modularité, la maintenabilité et l’évolutivité du projet.



6. Conclusion
l'application combine planification intelligente, analyse de performance et gamification dans une application mobile moderne développée avec React Native.
Le projet vise à démontrer des compétences techniques en développement mobile tout en proposant une solution utile pour l’apprentissage et l’amélioration personnelle
